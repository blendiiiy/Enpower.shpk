# Foto për Enpower

Shtoni fotot tuaja këtu. Next.js i optimizon automatikisht.

## Struktura

- **hero/** – imazh për seksionin kryesor (p.sh. `banner.jpg`, 1200×700 px)
- **about/** – foto për "Rreth nesh" (p.sh. `about.jpg`, 800×600 px)
- **projects/** – galeri projektesh (p.sh. `1.jpg`, `2.jpg`, `3.jpg`, 600×400 px)

Vetëm vendosni skedarët në këto foldera dhe përdorni path si `/images/hero/banner.jpg` në kod.
